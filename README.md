# Humam Ahmed Izu — Portfolio

A minimal, AVA SRG–inspired portfolio for Software Quality Assurance Engineer 2.

## Theme

- **Fonts:** Syne (headings), Space Grotesk (body) — similar to the [AVA SRG](https://srg.ava-digital.site/en) aesthetic
- **Colors:** Dark background (`#0a0a0a`), light text, accent `#d4a574` used for **Izu** and key UI elements
- **Sections:** Hero, About, Experience (previous companies), Contact

## Customize

1. **Experience:** Edit the company cards in `index.html` (company name, dates, role, description).
2. **Contact:** Replace `your@email.com` and LinkedIn/GitHub URLs with your real links.
3. **Accent color:** Change `--accent` in `styles.css` if you want a different highlight for "Izu".

## Run locally

Open `index.html` in a browser, or use a simple server:

```bash
cd portfolio
python3 -m http.server 8080
```

Then visit http://localhost:8080
